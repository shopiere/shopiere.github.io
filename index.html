_Alternative uses for Github_ [Readwrite][1] article on seven alternative uses for Github includes writing. Besides the well-known github pages blogging and website creation tool, the writer JJ Merelo has written a sf book on github, taking edits from readers. Even if there is only one of you, github can provide a central place for all your writing... Also mentioned a traveller, Dylan Egan, who has planned a lengthy trip on Github. Subscribers can read his itinerary and edit files suggesting accomodations, restaurants, sights, etc. [1]: http://readwrite.com/2013/11/08/seven-ways-to-use-github-that-arent-coding#awesm=~on11gLiwgnBwTo
